This is an operating system's project.

Semaphores and Producer Consumer Problem are used to solve a real life problem


This project solves the Issue of wating and allocation of railway tracks to train.


Hence this is A Railway Platform Management System

Requirements:
1) Linux OS
2) Text Editor
3) GCC compoler

User should provide priorities and the arrival time of the trains along with the wait time. 
The Program will automatically arrange the high priority or early arriving train to platform 1 for the convenience of the passengers. The rest of the trains will come on their respective time in the subsequent stations.
